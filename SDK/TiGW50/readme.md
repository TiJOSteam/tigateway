# TiGW50 可编程4G模组

TiBOX-C200 是钛云物联开发的支持 4G Cat1 网络的 DTU 数传控制器, 它提供 1 个 RS485 和 1 个 RS232 端口，可通过目前的运营商 4G 网络将数据上报至云端,满足用户对设备的 接入需求。

## 文件目录说明



| 目录   | 说明                  |
| ------ | --------------------- |
| driver | USB驱动, 用于设备配置 |
| api    | API接口库             |
| help   | 相关文档              |
