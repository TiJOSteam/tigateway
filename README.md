# TiGW200 系列可编程4G边缘计算网关
TiGW200是钛云物联基于钛极OS(TiJOS)物联网操作系统开发的4G可编程数边缘计算网关， 用户可通过Java语言开发控制器内部的应用和控制逻辑以及与云平台交互过程。

## 适用产品

| 产品型号                                |                                  |
| --------------------------------------- | -------------------------------- |
| TiGW200可编程4G边缘计算网关             | 双路RS485                        |
| TiGW260可编程4G边缘计算网关(带音频输出) | 一路RS485, 支持音频输出，支持TTS |
|                                         |                                  |
